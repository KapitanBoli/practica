## Запуск проекта

### 1. Установка зависимостей

Установите зависимости с помощью Poetry:

```bash
poetry install
```

### 2. Запуск приложения

Запустите приложение:

```bash
python main.py
```

После запуска API будет доступно по адресу:

```
http://127.0.0.1:8000/docs
```

В Swagger UI доступна ручка для работы с нейронной сетью.

## Дообучение модели

Для дообучения нейронной сети используйте ноутбук:

```
easyocr-rus-train.ipynb
```

Пример датасета для обучения находится в директории:

```
dataset_v4/
```

## Оценка качества и производительности

Для оценки скорости работы и качества распознавания изображений используйте:

```bash
python loop.py
```

Для работы скрипта необходимы датасет и разметка, расположенные в директории:

```
worker/dataset_v2/
```
